Github Pinboard

Be proud of your contributors, and display them for all the world to see!

How To Use

  • Include the library: 
                <script src="http://copypastel.github.com/github-pinboard/lib/github-pinboard.js">
  • Create an element to hold the pinboard: 
                  <div id="node-pinboard"><div>
  • Create the pinboard: 
                  <script>
                /*  Only need the user and repo names are needed to display
                 *  a pinboard. containerId defaults to "github-pinboard".
                 */
                var user = "joyent", repo = "node", containerId = "node-pinboard"
                new GithubPinboard(user, repo, containerId); 
              </script>
  • Style to your liking: 
                  .github-pinboard-pin {
                background: black;
              }
